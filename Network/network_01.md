## 네트워크의 내부구조

-   **계층적 구조(Layered Architecture)**

## 네트워크의 역사

-   통신의 역사
    -   우편
    -   봉화
        -   최초의 전자기 신호(광통신, 디지털)를 이용한 통신 시스템
    -   전보(모스 부호)
        -   **Samuel F. B. Morse**가 1846년에 발명
        -   Data Network
        -   **Digital Network**
    -   전화
        -   **Graham Bell**에 의해 1876년 발명
        -   현대적 통신
        -   Voice Network
        -   **Analog Network**
        -   사용자의 폭발적 증가
        -   수동교환에서 자동교환으로 발전
        -   **DDD(Direct Distance Dialing)**라는 방식이 컴퓨터 네트워크에 많은 영향을 끼침
